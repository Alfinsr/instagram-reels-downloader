# Bulk Instagram Reels Downloader

**Created by [Alfinsr](https://github.com/Alfinsr)**

Script CLI untuk mendownload Instagram Reels secara massal (bulk) maupun satuan dengan tampilan yang rapi dan modern.

---

## Fitur

- Single Link Download
- Bulk Download dari file `.txt`
- Progress bar real-time + animasi
- Dukungan Cookies (sangat disarankan)
- Penamaan file otomatis: `Reels1. Caption.mp4`
- Error handling yang ramah
- Pengecekan cookies otomatis

---

### 1. Install Python
Download di [python.org](https://www.python.org/downloads/)
